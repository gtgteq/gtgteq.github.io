無料でドメインを取ったり，有料ドメイン取得サービスを比較したり．

無料ドメイン

自宅サーバなどで特に公開せずドメインだけが必要な場合，無料ドメインで十分．

nix

services.ddclient = {
  enable = true;
  domains = [ "___.mooo.com" ];
  server = "freedns.afraid.org";
  protocol = "freedns";
  interval = "360min";
  quiet = true;
  username = "";
  password = "";
};
